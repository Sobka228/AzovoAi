# Пошаговый гайд

### Шаг 1: Установка Python (если нет)

Проверьте:
```bash
python --version
```

Если версия < 3.8 или ошибка "not found" — скачайте с [python.org](https://www.python.org/downloads/)

### Шаг 2: Получите Bot Token

1. Откройте Telegram
2. Напишите [@BotFather](https://t.me/BotFather)
3. Команда: `/newbot`
4. Придумайте имя (например, `MyAwesiomeBot`)
5. Придумайте username (например, `my_awesome_bot`) — это должно быть уникально
6. Скопируйте токен из ответа BotFather

**Токен выглядит так:**
```
123456789:ABCdefGHIjklmnOPqrsTUVwxyz-1234567890
```

### Шаг 3: Клонируйте репо

Откройте терминал и выполните:

```bash
git clone https://github.com/YOUR_USERNAME/azovoai.git
cd azovoai
```

Или просто скачайте ZIP и распакуйте.

### Шаг 4: Создайте файл .env

Скопируйте шаблон:
```bash
cp .env.example .env
```

Откройте файл `.env` в редакторе (notepad, VS Code и т.д.):

```env
BOT_TOKEN=ВАШ_ТОКЕН_СЮДА
BOT_USERNAME=my_awesome_bot
ADMIN_USER_ID=7451061064
POLLINATIONS_MODEL=openai
RATE_LIMIT_SECONDS=2
PORT=10000
```

Вставьте ваш токен от BotFather вместо `ВАШ_ТОКЕН_СЮДА`.

**Пример (реальный токен не публикуйте!):**
```env
BOT_TOKEN=123456789:ABCdefGHIjklmnOPqrsTUVwxyz-1234567890
BOT_USERNAME=my_awesome_bot
```

### Шаг 5: Установите зависимости

```bash
pip install -r requirements.txt
```

Это установит 3 пакета:
- aiogram (Telegram bot фреймворк)
- requests (HTTP клиент)
- python-dotenv (для .env файлов)

### Шаг 6: Запустите бота

```bash
python main.py
```

Если всё работает, вы увидите:

```
✅ Bot started successfully
🤖 Model: openai (Pollinations.ai)
📁 Dataset: dataset.txt
```

### Шаг 7: Протестируйте

Откройте Telegram и:

1. Найдите своего бота (по username который вы указали)
2. Нажмите Start
3. Отправьте команду:
   ```
   /ping
   ```
4. Бот должен ответить с пингом, например:
   ```
   🏓 Bot Status
   Response time: 152 ms
   Uptime: 0:00:45
   Model: openai (Pollinations.ai)
   ```

**Все готово**

## Дальнейшее использование

### Общение с ботом

Просто пишите в бот:
```
привет азово, как дела?
```

Бот ответит случайным хамским ответом.

### Использование в группе

1. Добавьте бота в группу
2. Пишите с упоминанием:
   ```
   @my_awesome_bot напиши шутку
   ```
   
   или триггер слово:
   ```
   азово, что ты думаешь?
   ```

### Основные команды

```
/ping    - проверка статуса
/bot     - статистика
/help    - помощь
```

Полный список команд в [COMMANDS.md](GITHUB_COMMANDS.md)

### Остановить бота

В терминале нажмите `Ctrl + C` (или `Cmd + C` на Mac).

## Кастомизация

### Измените поведение

Откройте `main.py` и найдите `SYSTEM_PROMPT` (~455 строка).

**Сейчас:**
```python
SYSTEM_PROMPT = (
    "ты хам ебаный отвечай коротко и с маленькой буквы..."
)
```

**Измените на:**
```python
SYSTEM_PROMPT = (
    "Ты вежливый помощник. Отвечай коротко и по сути."
)
```

Сохраните и перезапустите бота.

### Добавьте свой триггер слово

В `main.py` найдите:

```python
TRIGGER_WORDS = ["азово", "azovo", f"@{BOT_USERNAME.lower()}"]
```

Добавьте:

```python
TRIGGER_WORDS = ["азово", "azovo", f"@{BOT_USERNAME.lower()}", "привет бот"]
```

Теперь бот ответит и на "привет бот".

### Измените задержку между ответами

В `.env`:

```env
RATE_LIMIT_SECONDS=2
```

Измените на:
- `1` для быстрого отклика
- `5` для медленного отклика

### Смените модель AI

В `.env`:

```env
POLLINATIONS_MODEL=openai
```

Измените на другую (смотри [API.md](GITHUB_API.md) для списка).

## Статистика

### Посмотреть статистику в боте

```
/bot
```

Покажет:
- Количество сообщений
- Активных пользователей
- Размер датасета
- Время работы

### Посмотреть датасет

```
/export_dataset
```

Скачается файл `exported_dataset.txt` с всеми сохранёнными сообщениями.

## Продвинутые трюки

### Планировщик автосообщений

1. Запустите:
   ```
   /thoughts_start
   ```

2. Добавьте тему:
   ```
   /thoughts_add
   ```
   Затем ответьте:
   ```
   напиши про яйца и батарейки
   ```

3. Установите время:
   ```
   /thoughts_time 20
   ```
   (сообщения будут в 20:00)

4. Смотрите список:
   ```
   /thoughts_list
   ```

5. Отправить сейчас:
   ```
   /thoughts_now
   ```

6. Остановить:
   ```
   /thoughts_stop
   ```

### Работа в группе

Правила:
1. Добавьте бота в группу
2. Дайте ему права администратора (для реакций)
3. Используйте триггеры:
   ```
   азово, напиши что-то смешное
   ```
   или
   ```
   @my_awesome_bot помощь
   ```

### Реакции на сообщения

Упомяните реакцию:
```
поставь 😂
```

Бот поставит эмодзи реакцию.
